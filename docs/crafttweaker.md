Allows detection of lunar and solar events.

## IWorld Expansion

You can call these methods on an IWorld instance:

| Method                         | Description                                 |
|--------------------------------|---------------------------------------------|
| boolean isHyxcateBloodMoon()   | Returns true if a blood moon is happening   |
| boolean isHyxcateBlueMoon()    | Returns true if a blue moon is happening    |
| boolean isHyxcateFullMoon()    | Returns true if a full moon is happening    |
| boolean isHyxcateStarShower()  | Returns true if a star shower is happening  |
| boolean isHyxcateGrimEclipse() | Returns true if a grim eclipse is happening |
| boolean isHyxcateRedGiant()    | Returns true if a red giant is happening    |
